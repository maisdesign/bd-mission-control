# bd-mission-control

A single-file, JARVIS-style mission control panel for [beads (`bd`)](https://github.com/steveyegge/beads) orchestration — drop one HTML file into any project and watch your bead graph live.

> 🚧 Under active development — this repo dogfoods itself: the `.beads/` tracker you see here is the demo data.

## Status

Bootstrapping. See `.beads/issues.jsonl` for the live plan.
